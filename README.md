** ---TIC TAC TOE--- ** 

2-player game, also locally known as ZERO-CROSS

Compile locally:-
1. g++ main.cpp(MIN GW)
2. ./a

## Clearing the Screen(For Aesthetics)
The game clears the terminal every turn to keep the board clean.Add in int main:-

```cpp
system("cls");   // Windows

system("clear"); // Linux / Mac
```
